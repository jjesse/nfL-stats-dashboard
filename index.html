Welcome to the NFL Stats Dashboard

Your comprehensive source for NFL statistics, schedules, and player performance data.

This dashboard provides real-time insights into the NFL season, including:

  • Schedule: View upcoming games with team records and venue information
  • Team Statistics: Compare team performance across the league
  • Player Leaders: Track top performers in passing, receiving, and rushing categories
  • Defensive Stats: View tackles, sacks, and interception leaders
  • Special Teams: Track kickers, punters, and return specialists
  • League Leaders: Quick view of top performers across all categories
  • Playoff Picture: Current playoff seeding and scenarios
  • Data Updates: Automatically updated statistics via GitHub Actions

Navigate using the menu above to explore different sections of the dashboard. All data is regularly updated to provide you with the most current information available.
